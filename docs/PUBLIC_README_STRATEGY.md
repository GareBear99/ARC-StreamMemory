# Public README Strategy

ARC-StreamMemory should be understood in under 10 seconds:

> Local-first visual second brain for AI-readable video, screen, snapshot, robotics, and source-spine memory.

The README targets high-intent searches:

- AI visual memory
- video memory for LLMs
- screen recording for AI
- local-first multimodal memory
- visual RAG
- deterministic video archive
- robotics camera memory
- FFmpeg frame sampling for AI

Public trust rules:

- Make complete features clear.
- Separate integration gates from finished features.
- Link every source/reference repo.
- Keep FFmpeg described as the current backend.
- Keep ARC-FusionCapture described as an adapter/future runtime layer.
