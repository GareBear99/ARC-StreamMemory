# Public Launch Post

I built **ARC-StreamMemory**: a local-first visual second brain for AI-readable video, screen recordings, screenshots, robotics camera feeds, and seeded source-spine memory.

It turns visual sessions into reproducible AI memory modules:

- regular FFmpeg video ingest
- snapshot-folder ingest
- AI frame-speed sampling
- SHA-256 frame hashes
- seeded source spine
- ARC-style receipts
- OmniBinary-style chunk maps
- Arc-RAR-style bundle manifests
- local viewer
- AI digest + module attachment JSON
