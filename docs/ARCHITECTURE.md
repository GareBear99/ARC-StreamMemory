# Architecture

ARC-StreamMemory converts visual sessions into deterministic memory objects.

```text
visual source
→ ingest policy
→ frame schedule
→ frame index
→ event timeline
→ OCR-ready index
→ AI digest
→ module attachment
→ ARC receipts
→ OmniBinary chunk map
→ Arc-RAR bundle manifest
→ memory spine
→ seed spine
→ viewer
```
